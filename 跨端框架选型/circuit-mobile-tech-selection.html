  • Ionic — Essentially WebView-based; lower UX ceiling

Recommendation

Cross-Platform First, Native as Fallback

React Native + Swift / Kotlin
React Native

Handles core business flows
AI chat, document viewer, search, etc.

+
Swift / Kotlin

Handles platform-level capabilities
Push notifications, camera, biometrics, etc.

React Native covers the primary business logic, ensuring development efficiency and access to a large talent pool.
Native code handles low-level platform capabilities and performance-critical modules, maximizing user experience.

Key Reasons for This Recommendation

✓ Cross-platform covers most business screens with higher efficiency
✓ Native fallback for system-level and performance-critical features
✓ React ecosystem — front-end teams can ramp up quickly
✓ Largest talent pool; easiest to hire for
✓ Performance fully sufficient for AI Q&A use cases
✓ Mature community with rich third-party library support
